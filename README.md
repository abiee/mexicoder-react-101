Mexicoder React 101
======================
Base project to play with React on the React course of Mexicoder.

What's inside
----------------
Batteries included:
 - Gulp
 - Browserify
 - Bootstrap
 - React
 - BrowserSync

Setup
-----
Clone the repository and install the dependencies.

    $ git clone https://github.com/abiee/mexicoder-react-101.git
    $ cd mexicoder-react-101
    $ npm install
    $ gulp serve

Do not forget to install globally gulp if not installed yet.
